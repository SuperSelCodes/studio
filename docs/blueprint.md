# **App Name**: RSTI Sense

## Core Features:

- Sensor Data Ingestion: Securely ingest raw sensor signal data from various IoT receivers for real-time processing.
- Spatial Data Processing Engine: Perform signal triangulation and apply advanced smoothing filters to raw data, generating high-fidelity spatial coordinates and confidence scores with sub-200ms latency.
- Real-time Asset Tracking Dashboard: Visualize asset movements on an interactive map or grid in real-time, offering instant updates and intuitive interaction for monitoring.
- Performance Metrics Display: Exhibit live key performance indicators, including tracking latency, positional precision, and system reliability, to validate PoC success criteria.
- AI-Powered Sensor Configuration Tool: A generative AI tool that suggests optimal sensor array placements and calibration parameters based on environmental analysis and desired accuracy, minimizing interference and maximizing coverage efficiency.
- Historical Data Archiving & Playback: Store processed spatial coordinate data in a scalable database, allowing for review, historical analysis, and replaying past asset movements.
- Anomaly Detection & Alerts: Identify and trigger real-time alerts for unexpected asset behaviors or deviations from predefined tracking patterns, leveraging confidence scores for improved accuracy.

## Style Guidelines:

- Color scheme: Dark theme, optimized for data-rich displays to reduce eye strain and highlight critical information.
- Primary color: A sophisticated and professional tech-blue (#2681CD), used for main interactive elements and key data visualization.
- Background color: A very dark charcoal with a subtle cool undertone (#1C2023), providing a high-contrast canvas for data.
- Accent color: A vibrant teal (#3CD1BF), reserved for highlighting real-time updates, alerts, and crucial interactive states.
- Font: 'Inter' (sans-serif) for both headlines and body text. Chosen for its exceptional readability across various screen sizes, modern aesthetic, and suitability for presenting technical data clearly.
- Icons: Use minimalist, crisp line icons to convey clarity and maintain a clean, technical interface consistent with precise data representation.
- Layout: A highly modular, grid-based dashboard approach that allows for efficient organization and clear hierarchy of real-time data feeds and performance metrics.
- Animation: Employ subtle, purposeful animations for data transitions, loading states, and user feedback, ensuring a smooth and responsive real-time user experience without distractions.
